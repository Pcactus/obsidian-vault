### 1. 索引文档
- [[00-索引/工作流总览]] - 完整的工作流设计文档
- [[00-索引/快速开始]] - 30 秒快速上手指南

### 2. 角色 1-战略规划工程师（增强版 v2.0）
**新增功能**：
- ✅ GitHub 热门竞品搜索（按 Star 数排序）
- ✅ SWOT 分析
- ✅ **亮点功能标注**（差异化优势识别）
- ✅ **推荐最适合深入分析的竞品**（综合评分系统）

**输出**：
- 《[产品类型] 战略规划报告》
- 竞品总览表
- 每个竞品的 SWOT 分析
- 亮点功能详解
- 推荐竞品及评分

### 3. 角色 2-部署与截图工程师（增强版 v2.0）
**新增功能**：
- ✅ **功能模块识别**（树状结构）
- ✅ **功能详情记录**（入口、步骤、元素、交互逻辑）
- ✅ **亮点功能验证**（对照角色 1 的报告进行验证）

**输出**：
- 《部署与截图报告》
- 《功能模块清单》
- 亮点功能验证报告

### 4. 角色 3-需求文档工程师（设计完成）
**职责**：基于竞品分析生成 PRD

**功能**：
- 产品定位设计
- 功能列表提炼（P0/P1/P2 优先级）
- 功能详细描述
- 交互流程设计
- 数据结构定义
- 迭代计划

**输出**：
- 《产品需求文档（PRD）》

### 5. 角色 4-原型工程师（设计完成）
**职责**：基于 PRD 生成可交互 HTML 原型

**功能**：
- 页面结构设计
- 原型实现（HTML/CSS/JS）
- 交互逻辑实现
- 原型标注
- 原型说明文档

**输出**：
- 可交互 HTML 原型
- 《原型说明文档》

## 工作流执行流程

```
用户输入产品类型
       ↓
┌─────────────────────────────────┐
│ 角色 1：战略规划工程师            │
│ - GitHub 搜索热门前 3            │
│ - SWOT 分析 + 亮点功能标注        │
│ - 推荐最适合深入分析的竞品       │
└─────────────────────────────────┘
       ↓
┌─────────────────────────────────┐
│ 角色 2：部署与截图工程师          │
│ - 自动部署竞品                   │
│ - 页面遍历 + 截图                │
│ - 功能模块识别（树状结构）        │
│ - 亮点功能验证                   │
└─────────────────────────────────┘
       ↓
┌─────────────────────────────────┐
│ 角色 3：需求文档工程师            │
│ - 产品定位设计                   │
│ - 功能列表提炼（P0/P1/P2）       │
│ - **全中文 PRD 文档生成**         │
└─────────────────────────────────┘
       ↓
┌─────────────────────────────────┐
│ 角色 4：原型工程师                │
│ - **全中文可交互 HTML 原型生成**   │
│ - 原型说明文档                   │
└─────────────────────────────────┘
       ↓
┌─────────────────────────────────┐
│ 人工审核                         │
│ - 确认后即可交付开发团队         │
└─────────────────────────────────┘
```

## 开放问题及解决方案

| 问题                             | 状态    | 解决方案                    |
| ------------------------------ | ----- | ----------------------- |
| 如果 GitHub 前 3 竞品都无法部署，备选方案是什么？ | ✅ 已解决 | 扩大搜索范围到前 10，或手动选择可部署的竞品 |
| 页面遍历的深度限制是多少？                  | ✅ 已解决 | 限制：最多 15 个页面，避免无限点击     |
| 功能模块的颗粒度如何定义？                  | ✅ 已解决 | 按页面路由 + 主要功能区域划分，输出树状结构 |
