## 交互说明

### [核心流程名称，如：任务管理流程]

1. 从 Dashboard 点击"[模块名称]"进入列表页
2. 点击"+ 新建"打开创建表单
3. 填写表单后点击"保存"，返回列表页
4. 点击条目进入详情页
5. 点击"编辑"按钮进入编辑页

### 其他交互

| 交互 | 说明 |
|------|------|
| 搜索 | 在列表页顶部搜索框输入关键词，实时过滤 |
| 筛选 | 点击筛选器，选择条件后点击"应用" |
| 排序 | 点击表头可切换升序/降序 |
| 删除 | 点击删除按钮，二次确认后执行 |
